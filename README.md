# FOOD-HUB

> Online Food Ordering system built using MERN stack.

## Usage

### Installation

- Clone the repository
- Run following commands to install the dependencies

### Frontend
Go to the project root directory and run the following commands

```
cd frontend
```
```
npm install
```


### Backend
Go to the project root directory and run the following commands

```
cd backend
```
```
npm install
```
